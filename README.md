### **Galeria de Imagens**

Este é um projeto de galeria de imagens, onde você pode visualizar uma lista de fotos e pesquisar por autor. O projeto utiliza a API pública Picsum Photos para exibir imagens aleatórias e permite que você filtre as fotos com base no nome do autor.

### **Tecnologias Utilizadas**

-**React**: Biblioteca JavaScript para criar interfaces de usuário.

-**Next.js**: Framework React para construção de aplicativos web.

-**Tailwind CSS**: Framework de CSS para design responsivo e eficiente.

-**React Icons**: Biblioteca para ícones, usada para o ícone da lupa na barra de pesquisa.

### **Funcionalidades**

-**Exibição de Imagens**: As imagens são carregadas dinamicamente a partir da API Picsum Photos.

-**Busca em Tempo Real**: Permite pesquisar imagens por autor à medida que o usuário digita.

-**Exibição de Mensagem de "Nenhuma foto encontrada"**: Quando não há resultados para a pesquisa.
